Future improvements

Help me improve this website!

There is a serie of future improvements that could be applied to "Secreto", these range from adding the ability to sign in the website, to simple things like refactoring some of the code to follow better practises. Here is a list of current ideas that might improve this website:

  • Sign in: Might be beneficial to add authorisation and authentication to this website.
  • Improve Dancing Men cipher: Right now the Dancing Men Cipher uses a font to satisfy the needs of encryption and decryption, however, it would be desirable to have all the images for each dancing man as clickable so they could populate the encrypted text to be decoded.
  • Refactor some of the code to follow better practices: Due to the time constraints of this project there are some code improvements that I would have liked to improve, for instance sub-dividing the pages in different folders in the repo, adding a wiki to the repo, or even just follow better name conventions
  • Add more visuals to the cipher pages: Would be beneficial to add extra data, for example, how many letters are in the input or output of a cipher
  • Add Contact me form, right now it's just university email but some users might benefit from just filling up a form
